Projects

  • Literacy Uplift

    Compatible web and mobile applications to help people learn literacy done by the George Brown Research and Innovation division. It uses Laravel for the webapp, and compatible Android and iPhone apps are coming. I cannot share the app before the official release.

  • Cisco Hackathon Chat Bot

    A chat bot integrating Github and Cisco Teams APIs allowing users to create chat rooms for each of their Github repositories, list all their repositories, list their collaborators, search github for repositories containing any term, and show their starred repositories.

  • Arduino Plant Monitoring Project

    A system to monitor plants, or the temperature, humidity, and light of any environment. Using an Arduino Mega, DH11 sensor, and photoresistor to monitor those variables. Sending sensor data over the web through a Raspberry Pi to be accessed anywhere incoming.
